## AI提示词：  

【注释】  
逐行注释下列规则/域名的功能和意义

【总结】  

重写下列注释，格式为“// 【精要总结...】+详细说明..."

【替换】  

将所有的“”替换成“”，并且每一行注释上方都空一行，例如  
（空行）  
// 注释1  
规则1  
（空行）  
// 注释2  
规则2
